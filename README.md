# Impact of Personal Audio Device Usage on Ear Health 

## Objective
To evaluate the association between PAD usage and ear health outcomes.

## Methods Used
- Data Cleaning in Excel and SAS
- PROC IMPORT
- PROC NPAR1WAY
- PROC UNIVARIATE
- PROC FREQ
- PROC MEANS
- PROC LOGISTIC
- Correlation Analysis

## Statistical Techniques
- Logistic Regression
- Descriptive Analysis
- Odds Ratio Estimation
- Hypothesis Testing(Chi square,Mann whitney U)
- p-value Interpretation
- Data Visualization

## Tools
- SAS
- Excel
- Google forms

## Key Findings
- Increased PAD usage was associated with higher odds of ear-related symptoms.
- Significant association observed (p < 0.05).
